# My Emacs Settings


## 说明
刚开始学习使用，比较low.



## 原则
* 注重效率.
* 学习.


## 用法
* `mkdir .emacs-backups` 在当前用户目录下创建`.emacs-backups`.
* `git clone` 本git仓库.
* `M-x list-packages` 下载`package-list.md`中的插件.
* 重启emacs.


## 功能

### python支持
主要是python语法支持和补全

#### 插件
* python
* elpy
* flycheck
* company-jedi
* anaconda-mode
* company-anaconda

### smex
* 快速输入命令

### ido
* 更好的打开文件

### markdown
* markdown语法支持
* 借助浏览器插件实现预览

### 其它
* 自己的键位设置
